# CFT_NotesApp
Приложение для заметок.
Для хранения заметок используется CoreData.
Возможно добавление фотографии из галереи или с телефона.
Удаление заметки происходит при: 
1) Свайпе вправо на странице с заметками и последующей нажатии кнопки "Удалить"; 
2) Всех пустых полях внутри заметки и при отсутствии фотографии.
